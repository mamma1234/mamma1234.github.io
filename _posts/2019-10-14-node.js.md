-   node
-   node -v
-   node helloworld.js

## package.json 모듈(종속성) 최신 버전 업데이트

-   npm 모듈 버전 확인
    npm show [모듈명] version

-   npm 모듈 개별 업데이트
    npm update [모듈명]

-   npm package.json 모듈을 한번에 업데이트 시켜주는 패키지
    npm install -g npm-check-updates

-   최신버전 확인
    ncu

-   최신버전 업데이트
    ncu -u
-   모듈 설치
    npm install
