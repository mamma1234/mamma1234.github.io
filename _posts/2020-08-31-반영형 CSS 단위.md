# em
- 타이포그래피에서 현재의 지정된 포인트 사이즈를 나타내는 단위
지금 폰트 사이즈를 나타내는 단위
부모의 font 사이즈를 곱ㄱ하는 값

- relative to parent element

```JavaScript

fontsize : 16px
1em == 16px

<div class="parent">
    Parent
    <div class ="child">Child</div>
</div>

.parent {
    font-size:8em; // 16px*8 = 128px
}
.child {
    font-size:0.5em; // 128px*0.5 = 64px
}
```

# ex
