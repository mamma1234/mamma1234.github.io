## 실행 파일 위치
which python3

## 파일자체와 모든 퍼미션은 그대로 두고 내용만 지우기
cat /dev/null > 파일이름

## disk usage. 디스크 사용 현황
du -sh */
### 옵션
-h : --human-readable
-s : --summarize. 전체 용량만 출력한다.
-a : --all. 모든 파일과 모든 하위 디렉토리도 출력한다.
-c : --total 마지막 줄에 합계를 출력한다.
-k : 출력 숫자의 단위를 1kb 로 지정한다.
-x : --one-file-system. 현재 파일시스템의 파일 사용양만 출력한
