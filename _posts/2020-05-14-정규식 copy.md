# Scraping


## Exception
    - 페이지를 혹은 서버를 찾을 수 없는 경우  : HTTPError
    - 존재하지 않는 객체의 태그에 접근한 경우 : AttributeError
```JavaScript
    try:
        html = urlopen(url)
    except HTTPError as e:
        e1 = e
        return e1
    
    try:
        bs0bj = BeautifulSoup(html.read(),'html.parser')
        title = bs0bj.body.h1
    except AttributeError as e:
        e2 = e
        return e2
```
