# Deno
Deno는 Node와 비슷한 문제를 푸는 JavaScript, 그리고 TypeScript를 실행하기 위한 새로운 Command-line Runtime입니다.

## 기반 기술
V8 JavaScript Runtime
Rust (C++를 대체)
Tokio (이벤트 루프로 사용)
TypeScript
